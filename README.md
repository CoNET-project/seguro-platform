# `seguro-platform` README

The Seguro Platform application.

## Install

```bash
yarn
```

## Build

### Web

```bash
yarn build:web
```

### Desktop

#### macOS

```bash
yarn build:desktop:mac
```

#### Windows

```bash
yarn build:desktop:win
```

#### Linux

```bash
yarn build:desktop:lin
```

## Develop

```bash
yarn start
```

## Test

```bash
yarn test
```
