# `seguro-platform` README

The Seguro Platform application.

## Install

```bash
yarn
```

## Build

```bash
yarn build
```

## Run

```bash
serve -s build
```

View the production build at http://localhost:5000.

## Develop

```bash
yarn start
```

View the dev server at http://localhost:3000.

## Test

```bash
yarn test
```
